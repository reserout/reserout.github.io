Определение функций

DEF ИМЯ_ФУНКЦИИ(ПАРАМЕТРЫ)
    ОПРЕДЕЛЕНИЕ_ФУНКЦИИ
**
def hello():
    print("Hello world")   - функция создана,но не вызвана

hello()   - теперь функция вызвана (в скобках задаем параметры для функции)
hello()
hello()   - можно несколько раз вызывать одну и ту же функцию
***
Теперь попробуем задать функции параметры:

x = int(input("Введите 1 число:"))
y = int(input("Введите 2 число:"))

def sum(a,b): - имена переменных даем другие чтоб они не сталкивались между собой
    return a + b  -сумируем данные переменных введенные пользователем

print(sum(x,y))  -выводим результат

Можно вывести результат другим способом:
Создаем еще одну переменную и выводим в ней результат

z = sum(x,y)
print(z)
